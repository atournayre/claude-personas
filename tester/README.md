# tester

Tests automatisés : UI Chrome, tests PHPUnit TDD, analyse de couverture.

## Skills

| Commande | Description |
|----------|-------------|
| `/tester:ui-test` | Tester automatiquement une interface utilisateur dans Chrome |

## Agents

| Agent | Rôle |
|-------|------|
| `test-analyzer` | Analyse les tests existants et la couverture |
| `tester` | Écrit les tests PHPUnit en approche TDD |

## Installation

```bash
cp -r tester ~/.claude/plugins/tester
```
