# Changelog - orchestrator

## [1.0.0] - 2026-02-19

### Added
- Skill `orchestrator:feature` : workflow complet de développement en 8 phases (interactif) + mode automatique via issue GitHub (10 phases)
- Skill `orchestrator:check-prerequisites` : vérification des prérequis système avant workflow automatique
- Skill `orchestrator:fetch-issue` : récupération et validation d'une issue GitHub
- Skill `orchestrator:parallel` : implémentation parallèle de N issues via worktrees isolés
- Skill `orchestrator:ralph` : setup du loop autonome de développement Ralph
- Skill `orchestrator:status` : affichage de l'état courant du workflow de développement
- Skill `orchestrator:team` : orchestration d'équipes d'agents spécialisés pour tâches complexes
- Skill `orchestrator:transform` : transformation de prompts en prompts exécutables avec Task Management System
- Skill `orchestrator:validate` : validation de checklist avant exécution
