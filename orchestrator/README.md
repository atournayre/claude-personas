# orchestrator

Orchestrateur de workflows de développement : feature complète, mode automatique, implémentation parallèle, équipes d'agents.

## Skills

| Commande | Description |
|----------|-------------|
| `/orchestrator:check-prerequisites` | Vérifier les prérequis avant développement |
| `/orchestrator:feature` | Workflow complet de développement de feature |
| `/orchestrator:fetch-issue` | Récupérer le contenu d'une issue GitHub |
| `/orchestrator:parallel` | Implémentation parallèle avec worktrees |
| `/orchestrator:ralph` | Configurer la boucle autonome Ralph |
| `/orchestrator:status` | Afficher le workflow et l'étape courante |
| `/orchestrator:team` | Orchestrer une équipe d'agents spécialisés |
| `/orchestrator:transform` | Transformer un prompt en tâches structurées |
| `/orchestrator:validate` | Vérifier la checklist avant exécution |

## Installation

```bash
cp -r orchestrator ~/.claude/plugins/orchestrator
```
