# architect <Badge type="info" text="v1.1.0" />

Design architectural, planification d'implémentation et documentation des décisions techniques.

## Installation

```bash
/plugin install architect@atournayre
```

## Skills Disponibles

Le plugin architect fournit 4 skills :

### `/architect:design`

Designer des approches architecturales et choisir la meilleure — Phase 3 du workflow.

### `/architect:plan`

Générer un plan d'implémentation détaillé dans `docs/specs/` — Phase 4 du workflow.

### `/architect:adr`

Génère un Architecture Decision Record (ADR) formaté et structuré.

### `/architect:start`

Démarre un développement avec un starter léger puis active le mode plan.

## Agents

- **architect** : Analyse l'architecture et propose une conception technique solide
- **challenger** : Avocat du diable — challenge les propositions, identifie les edge cases et failles de sécurité
- **deep-think** : Délègue les problèmes complexes à Gemini Deep Think pour une réflexion approfondie
