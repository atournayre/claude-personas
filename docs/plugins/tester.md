# tester <Badge type="info" text="v1.1.0" />

Tests automatisés : UI Chrome, tests PHPUnit TDD, analyse de couverture.

## Installation

```bash
/plugin install tester@atournayre
```

## Skills Disponibles

Le plugin tester fournit 1 skill :

### `/tester:ui-test`

Teste automatiquement une interface utilisateur dans Chrome : navigation, clics, validation visuelle, tests fonctionnels, responsive, comportement et debug. Génère un rapport détaillé avec screenshots et optionnellement un GIF du parcours.

## Agents

- **tester** : Écrit les tests PHPUnit en approche TDD
- **test-analyzer** : Analyse la couverture et qualité des tests PHPUnit dans une PR
