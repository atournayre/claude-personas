# implementer <Badge type="info" text="v1.1.0" />

Skills d'implémentation de code : code, debug, refactoring, correction d'issues et PR.

## Installation

```bash
/plugin install implementer@atournayre
```

## Skills Disponibles

Le plugin implementer fournit 8 skills :

### `/implementer:code`

Implémente selon le plan — avec ou sans approbation préalable.

### `/implementer:debug`

Analyse et résout une erreur (message simple ou stack trace).

### `/implementer:oneshot`

Implémentation ultra-rapide d'une feature ciblée — Explore, Code, Test.

### `/implementer:refactor`

Refactoring autonome piloté par les tests — boucle refactor/test/ajust.

### `/implementer:implement`

Force l'implémentation complète d'une feature (pas juste planification).

### `/implementer:fix-issue`

Corrige une issue GitHub avec workflow structuré et efficace.

### `/implementer:fix-pr-comments`

Récupère les commentaires de review PR et implémente tous les changements demandés.

### `/implementer:log`

Ajoute des fonctionnalités de log à une classe PHP (LoggableInterface).

## Agents

- **developer** : Implémente le code de production selon le plan validé
- **implementer** : Implémente le code ET écrit les tests selon le plan validé
