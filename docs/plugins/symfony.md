# symfony <Badge type="info" text="v1.0.0" />


Expert Symfony couvrant le framework core et l'ecosysteme officiel : Doctrine, Twig, Security, Messenger, Mailer, Form, Console et 150+ bundles.

## Skills

| Commande | Description |
|----------|-------------|
| `/symfony:console` | Generer une commande console Symfony |
| `/symfony:controller` | Generer un controleur avec routing attributs |
| `/symfony:form` | Generer un FormType avec contraintes de validation |
| `/symfony:event` | Generer un Event + EventSubscriber/Listener |
| `/symfony:message` | Generer un Message Messenger + Handler |
| `/symfony:voter` | Generer un Voter de securite |
| `/symfony:mailer` | Generer un email (TemplatedEmail / Notification) |
| `/symfony:config` | Assistant de configuration de bundles Symfony |
| `/symfony:diagnose` | Diagnostiquer les problemes d'une application Symfony |

## Agents

| Agent | Description |
|-------|-------------|
| `symfony-expert` | Expert generaliste Symfony pour questions d'architecture et bonnes pratiques |
| `security-auditor` | Auditeur de securite de la configuration Symfony |

## Installation

```bash
cp -r symfony ~/.claude/plugins/symfony
```

## Dependances optionnelles

- `php` — pour la generation d'entites Elegant Objects
- `twig` — pour la generation de Twig Components
