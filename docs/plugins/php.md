# php <Badge type="info" text="v1.1.0" />

Skills PHP/Symfony orientés Elegant Objects : génération d'entités, collections, contrats, factories, stories.

## Installation

```bash
/plugin install php@atournayre
```

## Skills Disponibles

Le plugin php fournit 9 skills :

### `/php:make-all`

Génère tous les fichiers pour une entité complète (orchestrateur).

### `/php:make-entity`

Génère une entité Doctrine avec repository selon les principes Elegant Objects.

### `/php:make-collection`

Génère une classe Collection typée avec traits Atournayre.

### `/php:make-contracts`

Génère les interfaces de contrats pour une architecture Elegant Objects.

### `/php:make-invalide`

Génère une classe Invalide (exceptions métier).

### `/php:make-out`

Génère une classe Out (DTO immuable pour output).

### `/php:make-factory`

Génère une Factory Foundry pour les tests.

### `/php:make-story`

Génère une Story Foundry pour les fixtures de tests.

### `/php:make-urls`

Génère une classe Urls + Message CQRS + Handler.
