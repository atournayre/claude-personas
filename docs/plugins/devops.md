# devops <Badge type="info" text="v1.1.2" />


Opérations DevOps Git : branches, commits, worktrees, PR, CI, releases.

## Skills

| Commande | Description |
|----------|-------------|
| `/devops:branch` | Créer une branche Git avec workflow structuré |
| `/devops:cd-pr` | Créer une PR en mode Continuous Delivery |
| `/devops:ci-autofix` | Corriger automatiquement les erreurs CI |
| `/devops:commit` | Créer un commit conventionnel avec emoji |
| `/devops:conflict` | Analyser et résoudre les conflits Git |
| `/devops:pr` | Créer une Pull Request GitHub standard |
| `/devops:release-notes` | Générer des notes de release orientées utilisateurs |
| `/devops:release-report` | Générer un rapport d'impact entre deux branches |
| `/devops:worktree` | Gérer les git worktrees |

## Installation

```bash
cp -r devops ~/.claude/plugins/devops
```
