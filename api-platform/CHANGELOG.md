# Changelog

## [1.0.0] - 2026-02-25

### Added
- Initial release
- `api-platform:make-resource` — Génération d'API Resource complète avec opérations, sérialisation et validation
- `api-platform:make-filter` — Génération de filtres (SearchFilter, OrderFilter, DateFilter, RangeFilter, custom)
- `api-platform:make-provider` — Génération de State Provider pour la récupération de données
- `api-platform:make-processor` — Génération de State Processor pour la persistence
- `api-platform:make-dto` — Génération de DTOs input/output avec providers et processors associés
- `api-platform:make-test` — Génération de tests fonctionnels (ApiTestCase)
- `api-platform:audit` — Audit de bonnes pratiques, sécurité et performances
- Agent `api-expert` — Expert architecture API Platform pour analyse technique
- Agent `reviewer` — Revue de code spécialisée API Platform
