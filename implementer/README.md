# implementer

Skills d'implémentation de code : code, debug, refactoring, correction d'issues et PR.

## Skills

| Commande | Description |
|----------|-------------|
| `/implementer:code` | Implémenter selon le plan |
| `/implementer:debug` | Analyser et résoudre une erreur |
| `/implementer:fix-issue` | Corriger une issue GitHub |
| `/implementer:fix-pr-comments` | Implémenter les commentaires de review PR |
| `/implementer:implement` | Implémenter le code et les tests |
| `/implementer:log` | Ajouter des fonctionnalités de log |
| `/implementer:oneshot` | Implémentation rapide Explore → Code → Test |
| `/implementer:refactor` | Refactoriser le code |

## Agents

| Agent | Rôle |
|-------|------|
| `developer` | Implémente le code selon le plan validé |
| `implementer` | Implémente le code ET écrit les tests |

## Installation

```bash
cp -r implementer ~/.claude/plugins/implementer
```
